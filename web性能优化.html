webжҖ§иғҪдјҳеҢ–

еҸ‘иЎЁдәҺ 2021-06-09

жҖ§иғҪжҳҜз”ЁжҲ·дҪ“йӘҢзҡ„дёҖдёӘйҮҚиҰҒж–№йқўгҖӮзҪ‘йЎөзҡ„еҠ иҪҪйҖҹеәҰе’Ңе“Қеә”з”ЁжҲ·иҫ“е…Ҙзҡ„йҖҹеәҰдјҡжҳҫи‘—еҪұе“Қз”ЁжҲ·еҜ№жӮЁзҪ‘з«ҷзҡ„зңӢжі•пјҢе№¶дё”иҝҳдјҡеҶіе®ҡз”ЁжҲ·жҳҜз»§з»ӯжөҸи§ҲжӮЁзҡ„зҪ‘з«ҷиҝҳжҳҜж”ҫејғжӮЁзҡ„зҪ‘з«ҷгҖӮ

еүҚз«ҜжҖ§иғҪдјҳеҢ–зі»з»ҹеҲҶжһҗ

  • жҖ§иғҪдјҳеҢ–зҡ„ж•ҙдҪ“жҖқи·Ҝ
  • йқўиҜ•иҝҮзЁӢдёӯжҖҺд№ҲжӣҙеҘҪзҡ„еӣһзӯ”

дёәд»Җд№ҲжҖ§иғҪдјҳеҢ–еҰӮжӯӨйҮҚиҰҒ

  • жҖ§иғҪеҜ№з”ЁжҲ·зҡ„еҪұе“Қ
  • з•ҷеӯҳзҺҮ
  • иҪ¬еҢ–зҺҮ
  • е“ҒзүҢеҪўиұЎ
  • з”ЁжҲ·ж»Ўж„ҸеәҰ
    д»ҺиҝҷдёӘзҪ‘з«ҷеҸҜд»Ҙзі»з»ҹеӯҰд№ жҖ§иғҪдјҳеҢ–ж–№йқўзҡ„зҹҘиҜҶпјҡweb.dev[https://web.developers.google.cn/]

д»Һиҫ“е…Ҙ URL еҲ°йЎөйқўеҠ иҪҪе®ҢжҲҗ дёӯй—ҙеҸ‘з”ҹдәҶд»Җд№Ҳпјҹ

  1. еӨ„зҗҶURL

    • жөҸи§ҲеҷЁеҲӨж–ӯжҳҜжҗңзҙўиҜҚиҝҳжҳҜеҗҲжі•URL
    • иЎҘе…ЁеҚҸи®®пјҢй»ҳи®ӨHTTPS
    • жЈҖжҹҘ HSTSпјҢпјҲHSTSжҳҜHTTPдёҘж јдј иҫ“е®үе…ЁпјҲHTTP Strict Transport Securityпјүзҡ„зј©еҶҷпјү

  2. жң¬ең°ејәзј“еӯҳеҲӨж–ӯ

    • е…ҲжҹҘ Memory Cache (еҶ…еӯҳзј“еӯҳ)
    • еҶҚжҹҘ Disk Cache (зЈҒзӣҳзј“еӯҳ)
    • дҫқжҚ®пјҡCache-control: max-age=xxx(и®ҫзҪ®иө„жәҗеңЁжң¬ең°зј“еӯҳзҡ„жңүж•Ҳж—¶й—ҙ[з§’]пјҢвҖҢдјҳе…Ҳзә§жңҖй«ҳвҖҢвҖҢ), Expires (HTTP/1.0 з»қеҜ№иҝҮжңҹж—¶й—ҙпјҢиӢҘдёҺ Cache-Control еҗҢж—¶еӯҳеңЁпјҢвҖҢд»Ҙ max-age дёәеҮҶвҖҢвҖҢ)
    • е‘Ҫдёӯ -> дҪҝз”Ёжң¬ең°иө„жәҗдёҚиө°зҪ‘з»ңпјҢзӣҙжҺҘиҝӣе…Ҙи§ЈжһҗжёІжҹ“жөҒзЁӢ
    • жңӘе‘Ҫдёӯ -> з»§з»ӯеҫҖдёӢиө°зҪ‘з»ңиҜ·жұӮ

  3. DNSи§Јжһҗ(еҹҹеҗҚ -> IPең°еқҖ)

    • жөҸи§ҲеҷЁзј“еӯҳвҖҢпјҡжЈҖжҹҘиҝ‘жңҹи§Јжһҗз»“жһң
    • ж“ҚдҪңзі»з»ҹзј“еӯҳвҖҢпјҡжҹҘиҜў hosts ж–Үд»¶
    • жң¬ең° DNS жңҚеҠЎеҷЁпјҲLDNSпјүвҖҢпјҡз”ұ ISP(еӣҪеҶ…дёүеӨ§иҝҗиҗҘе•Ҷд№ӢжөҒ) жҸҗдҫӣпјҢйҖҡеёёиғҪе®ҢжҲҗ 80% зҡ„и§Јжһҗ
    • ж №еҹҹеҗҚжңҚеҠЎеҷЁ вҶ’ йЎ¶зә§еҹҹпјҲеҰӮ .comпјүвҶ’ жқғеЁҒеҹҹеҗҚжңҚеҠЎеҷЁвҖҢпјҡйҖҗзә§жҹҘиҜўзӣҙиҮіиҺ·еҫ— IP ең°еқҖгҖӮ

      жӯӨиҝҮзЁӢйҒөеҫӘвҖңзј“еӯҳдјҳе…ҲгҖҒйҖҗеұӮйҖ’еҪ’вҖқеҺҹеҲҷпјҢиҖ—ж—¶йҖҡеёёеңЁеҮ жҜ«з§’еҲ°ж•°зҷҫжҜ«з§’д№Ӣй—ҙ вҖҢ

  4. е»әз«ӢTCPиҝһжҺҘ

    • жөҸи§ҲеҷЁдҪҝз”Ёи§Јжһҗеҫ—еҲ°зҡ„ IP ең°еқҖдёҺжңҚеҠЎеҷЁе»әз«Ӣ TCP иҝһжҺҘгҖӮ
    • йҖҡиҝҮ вҖҢдёүж¬ЎжҸЎжүӢвҖҢ зЎ®и®ӨиҝһжҺҘпјҡ
      1. е®ўжҲ·з«ҜеҸ‘йҖҒ SYN еҢ…пјӣ
      2. жңҚеҠЎеҷЁеӣһеӨҚ SYN+ACK еҢ…пјӣ
      3. е®ўжҲ·з«ҜеҸ‘йҖҒ ACK еҢ…пјҢиҝһжҺҘе»әз«Ӣ вҖҢ

        иӢҘдҪҝз”Ё HTTPSпјҢиҝҳйңҖйўқеӨ–иҝӣиЎҢ вҖҢTLS жҸЎжүӢвҖҢпјҲеҚҸе•ҶеҠ еҜҶз®—жі•гҖҒдәӨжҚўеҜҶй’ҘзӯүпјүпјҢд»Ҙе»әз«Ӣе®үе…ЁйҖҡйҒ“

  5. еҸ‘йҖҒ http иҜ·жұӮпјҢ еёҰдёҠеҚҸе•Ҷзј“еӯҳеӨҙ(ETag + If-None-Match)

    • жөҸи§ҲеҷЁжһ„йҖ  HTTP иҜ·жұӮжҠҘж–ҮпјҢеҢ…еҗ«пјҡ
      • вҖҢиҜ·жұӮиЎҢвҖҢпјҡж–№жі•пјҲеҰӮ GETпјүгҖҒи·Ҝеҫ„гҖҒеҚҸи®®зүҲжң¬пјӣ
      • иҜ·жұӮеӨҙвҖҢпјҡеҰӮ HostгҖҒUser-AgentгҖҒAccept-EncodingгҖҒCookie зӯү
      • иҜ·жұӮдҪ“вҖҢпјҲд»… POST/PUT зӯүж–№жі•жңүпјүгҖӮ

  6. жңҚеҠЎеҷЁеӨ„зҗҶе№¶иҝ”еӣһе“Қеә”

    • жңҚеҠЎеҷЁжҺҘж”¶иҜ·жұӮпјҢз”ұ Web жңҚеҠЎпјҲеҰӮ NginxгҖҒApacheпјүеӨ„зҗҶгҖӮ
    • ж №жҚ®иҜ·жұӮи·Ҝеҫ„и°ғз”ЁеҜ№еә”зЁӢеәҸпјҲеҰӮ PHPгҖҒNode.jsгҖҒjavaпјүпјҢз”ҹжҲҗе“Қеә”гҖӮ
    • 304 -> дҪҝз”Ёжң¬ең°зј“еӯҳ
    • 200 -> иҝ”еӣһж–° HTML
    • е“Қеә”еҢ…еҗ«пјҡ
      • зҠ¶жҖҒз ҒвҖҢпјҲеҰӮ 200 OKгҖҒ301 Moved PermanentlyгҖҒ404 Not Foundпјүпјӣ
      • е“Қеә”еӨҙвҖҢпјҡеҰӮ Content-TypeгҖҒCache-ControlгҖҒSet-Cookieпјӣ
      • вҖҢе“Қеә”дҪ“вҖҢпјҡйҖҡеёёжҳҜ HTML ж–ҮжЎЈ

  7. жһ„е»әжёІжҹ“ж ‘е№¶еёғеұҖз»ҳеҲ¶

    • еҗҲе№¶ DOM е’Ң CSSOMпјҢз”ҹжҲҗ вҖҢжёІжҹ“ж ‘пјҲRender TreeпјүвҖҢпјҢд»…еҢ…еҗ«еҸҜи§ҒиҠӮзӮ№гҖӮ
    • и®Ўз®—жҜҸдёӘиҠӮзӮ№зҡ„еҮ дҪ•дҪҚзҪ®е’Ңе°әеҜёпјҲвҖҢеёғеұҖ/LayoutвҖҢпјүгҖӮ
    • е°ҶжёІжҹ“ж ‘иҪ¬жҚўдёәеғҸзҙ пјҲвҖҢз»ҳеҲ¶/PaintвҖҢпјүпјҢе№¶еҲҶеұӮеӨ„зҗҶпјҲеҰӮе®ҡдҪҚе…ғзҙ гҖҒйҖҸжҳҺеұӮзӯүпјүвҖҢ

  8. еҠ иҪҪеӯҗиө„жәҗдёҺдәӨдә’з»‘е®ҡ

    • и§Јжһҗ HTML дёӯзҡ„ <script>гҖҒ<img>гҖҒ<link> зӯүж ҮзӯҫпјҢвҖҢйҖ’еҪ’еҸ‘иө·иҜ·жұӮвҖҢеҠ иҪҪ CSSгҖҒJSгҖҒеӣҫзүҮзӯүиө„жәҗгҖӮ
    • жү§иЎҢ JavaScriptпјҲеҸҜиғҪдҝ®ж”№ DOM/CSSOMпјҢи§ҰеҸ‘йҮҚжҺ’йҮҚз»ҳпјүгҖӮ
    • йЎөйқўе®Ңе…ЁеҠ иҪҪеҗҺпјҢи§ҰеҸ‘ DOMContentLoaded е’Ң load дәӢд»¶пјҢз”ЁжҲ·еҸҜдёҺйЎөйқўдәӨдә’ вҖҢ

еҸӘжңүж·ұеҲ»зҗҶи§ЈдәҶдёҖдёӘзҪ‘йЎөд»Һ URL еҲ°йЎөйқўеұ•зӨәзҡ„ж•ҙдёӘиҝҮзЁӢдёӯеҸ‘з”ҹдәҶд»Җд№ҲпјҢжҲ‘д»¬жүҚиғҪзҹҘйҒ“еҒҡдјҳеҢ–зҡ„ж—¶еҖҷпјҢеҸҜд»Ҙд»Һе“Әдәӣж–№йқўе…ҘжүӢ

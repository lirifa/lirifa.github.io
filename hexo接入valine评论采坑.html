hexo接入valine评论踩坑

发表于 2020-11-17 更新于 2020-11-18 阅读次数： Valine：

配置过程很简单
1、 注册leanCloud，创建应用
2、 按照教程配置好AppId和AppKey

然后问题来了：

Code 401: 未经授权的操作，请检查你的AppId和AppKey.

折腾了一个下午，还是没有找到问题所在，后来注册了leanCloud国际版重试了一遍，好了

划重点：一定要注册 leanCloud国际版
